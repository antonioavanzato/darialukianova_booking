# Иконки PWA

Положите сюда три PNG-файла (на чёрном фоне `#1d1d1f`, минималистичная монограмма
в стиле сайта Дарьи):

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512
- `icon-512-maskable.png` — 512×512, с безопасными полями (maskable, ~20% padding)

Имена должны совпадать с указанными в `manifest.json`.
